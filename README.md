

Guide-for-10th
-----------------
Guide-for-10th is a full-stack educational web application built with the MERN stack (MongoDB, Express.js, React, Node.js). 
It provides students with easy access to study guides, resources, and a chatbot for interactive learning. The backend is hosted on 
Render, and the frontend is hosted on Vercel.

Features
-----------
Authentication

1. User registration and login with JWT-based authentication.
2. Passwords are hashed using bcryptjs.
3. Secure cookies for maintaining sessions.
4. Logout functionality.

User Profile
-----------

1. View and update user profile data.
2. Track class, age, and interests.

Chatbot
--------
1. Interactive AI-based chatbot for answering educational queries.
2. Chat history management (optional for future versions).
3. Admin Functionality (Optional)
4. Admin login with environment-based credentials.
5. Access to manage users and content.

CORS & Deployment Ready
-----------------------

1. Backend configured to allow requests from both local dev and Vercel frontend.
2. Cookies configured for cross-origin requests with HTTPS.
